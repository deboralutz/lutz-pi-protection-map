# Identidade Visual

## Objetivo

Registrar fundamentos visuais do projeto: paleta, tipografia, iconografia, ritmo de layout e criterios de acessibilidade.

## Direcao inicial

- Interface clara, precisa e institucional.
- Hierarquia visual orientada a diagnostico e tomada de decisao.
- Componentes consistentes para estados, alertas, niveis de risco e recomendacoes.
